# Dongle

Cracker setup based on ***Dongle project*** **(previous version)**
for ***Optitex Pattern Design*** **(Sentinel)** made in ***Qt/C++***

> ### Program stages
> 1. ***Installation***: <br>
>     - Install **required files**
>     - **Add second stage** to startup programs
> 2. ***Finalize after restart***:
>     - Install drivers
